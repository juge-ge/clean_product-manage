## 二、清洁生产管理模块扩展

### 2.1 模块结构
```
清洁生产管理
├── 清洁生产政策与管理
│   ├── 通知公告
│   ├── 动态信息
│   ├── 清洁生产法规政策
│   ├── 能力建设
│   └── 会议与宣传

├── 清洁生产审核典型案例
```

```javascript
// api/modules/clean-production/policy.js
export const policyApi = {
  // 通知公告
  getNoticeList: (params) => request.get('/api/v1/policy/notice', { params }),

  updateNotice: (id, data) => request.put(`/api/v1/policy/notice/${id}`, data),
  deleteNotice: (id) => request.delete(`/api/v1/policy/notice/${id}`),
  
  // 动态信息
  getNewsList: (params) => request.get('/api/v1/policy/news', { params }),
```
